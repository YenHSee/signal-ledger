# frontend-web

React + Vite + Tailwind UI: multi-factor screener and stock detail page
(AI report, fundamentals, 30-day price chart with AI/news markers, news feed).

See the [root README](../../README.md) for the full project overview.

## Run locally

```bash
pnpm install   # from repo root
pnpm dev       # http://localhost:5173
```

Expects the backend at `http://localhost:4000/api`.
